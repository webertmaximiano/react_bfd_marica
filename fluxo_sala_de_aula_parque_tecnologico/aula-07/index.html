2. Hexadecimal

Propriedades de Fontes no CSS

font-family: Define a família da fonte (ex: serif, sans-serif).

font-size: Altera o tamanho do texto (ex: 16px, 1.2em).

font-weight: Define o "peso" ou a "grossura" da fonte (ex: normal, bold).

font-style: Define o estilo da fonte (ex: italic).

text-decoration: Adiciona linhas decorativas (ex: underline, line-through).

text-transform: Muda o case do texto (ex: uppercase, capitalize).

line-height: Define a altura da linha, controlando o espaçamento entre as linhas de um parágrafo. Isso melhora a legibilidade.

letter-spacing: Aumenta ou diminui o espaço entre as letras.

word-spacing: Aumenta ou diminui o espaço entre as palavras.

text-align: Alinha o texto horizontalmente (left, center, right, justify).

font (shorthand): Uma propriedade "atalho" para definir vários valores de fonte de uma só vez.

Propriedades de Margin

A propriedade margin é usada para criar espaço AO REDOR dos elementos, fora de qualquer borda definida, parte externa.

Eu tenho uma margem de 10px no topo (margin-top).

Eu tenho uma margem de 20px na direita (margin-right).

Eu tenho uma margem de 30px embaixo (margin-bottom).

Eu tenho uma margem de 40px na esquerda (margin-left).

Eu uso a propriedade "atalho" (shorthand) para definir todas as margens de uma vez: 10px 20px 30px 40px.

Propriedades de Padding

A propriedade padding é usada para criar espaço INTERNO em um elemento, entre o conteúdo e a sua borda.

Eu tenho um preenchimento de 10px no topo (padding-top).

Eu tenho um preenchimento de 20px na direita (padding-right).

Eu tenho um preenchimento de 30px embaixo (padding-bottom).

Eu tenho um preenchimento de 40px na esquerda (padding-left).

Eu uso a propriedade "atalho" (shorthand) para definir todos os preenchimentos de uma vez: 10px 20px 30px 40px.

Propriedades de Border

A propriedade border é usada para criar uma linha ao redor de um elemento, entre o padding e a margin.

border-width: Define a espessura da borda.

border-style: Define o estilo da linha (solid, dotted, dashed, etc.).

border-color: Define a cor da borda.

Podemos aplicar propriedades a apenas um lado, como em border-left.

border-radius: Arredonda os cantos da borda.

border (shorthand): A forma abreviada para configurar espessura, estilo e cor de uma só vez.
